How to get rid of useless kernels

under Ubuntu







Introduction

About the author

Introduction

Lots of kernel updates come along with the regular updates of the distribution. In order to avoid disk saturation, it is a good rule of thumb to get rid of the oldest version.

Recipe

Get the version of the current running kernel:

$ uname -r
2.6.22-14-generic

List the installed linux images (the current versions are in red):

$ dpkg -l | grep linux-image
